# FC3.0-Hud
Hud inspired on 5City
(Update 1.5 is already out)

# Dependecies
- ESX Framework
- esx_status
- pma-voice
- Stress System by utku (https://github.com/utkuali/Stress-System-by-utku)
- ox_lib

# Install
Copy all from pma.txt to pma-voice>client>module>radio.lua
